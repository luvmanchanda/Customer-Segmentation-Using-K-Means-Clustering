# Customer Segmentation | K-Means Clustering Machine Learning Project

Developed an end-to-end Machine Learning project to segment customers into meaningful groups by implementing the complete ML workflow, including data collection, preprocessing, exploratory data analysis (EDA), model development, cluster visualization, and business insight generation.

The project utilized the Mall Customers dataset from Kaggle. Performed data cleaning, preprocessing, feature selection, and exploratory data analysis using **Annual Income** and **Spending Score** to identify customer segments through unsupervised machine learning techniques.

Implemented the **K-Means Clustering** algorithm and applied the **Within-Cluster Sum of Squares (WCSS)** with the **Elbow Method** to determine the optimal number of clusters, resulting in five well-defined customer segments. Created informative **scatter plot visualizations** to clearly illustrate the distribution of customers and cluster boundaries, making the segmentation results easy to interpret and analyze.

Derived actionable business insights by identifying high-value customers, moderate spenders, and low-spending customer groups, enabling personalized marketing strategies, targeted promotions, customer retention initiatives, and data-driven business decision-making.
